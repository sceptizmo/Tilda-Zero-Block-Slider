# Tilda Zero Block Slider

Скрипт для [Tilda](https://tilda.cc), который превращает zero-блоки (и любые другие блоки Тильды) в слайды. Переключение выполняется по клику на любые элементы с заданными классами «назад» / «вперёд».

## Возможности

- **Слайды из любых блоков**: zero-блоки и обычные блоки Тильды.
- **Свободный дизайн**: кнопки управления могут быть где угодно — в отдельном zero-блоке или внутри самих слайдов.
- **Несколько кнопок**: можно поставить несколько элементов управления на странице.
- **Свайп на мобильных**: включён для экранов до 960px.
- **Остановка медиа**: при смене слайда останавливает видео (HTML5 / YouTube / Vimeo).

## Установка

### Шаг 1. Назначьте классы слайдам

Добавьте класс **`uc-review-slide`** каждому блоку, который должен стать слайдом.

### Шаг 2. Назначьте классы элементам управления

- **`uc-review-prev`** — кнопка «назад»
- **`uc-review-next`** — кнопка «вперёд»

Элементы управления могут быть:
- в отдельном zero-блоке перед слайдами (удобно для стандартных блоков);
- внутри каждого слайда (любой кликабельный элемент).

### Шаг 3. Добавьте HTML-блок

1. Добавьте на страницу блок **«HTML-код»**
2. Скопируйте содержимое файла `embed.html` и вставьте в блок

## Как это работает

- Скрипт ищет все элементы с классом `uc-review-slide` и оборачивает их в общий контейнер.
- Клики по элементам `uc-review-prev` и `uc-review-next` переключают активный слайд.
- При смене слайда скрипт останавливает медиа внутри слайдера (HTML5-видео, YouTube, Vimeo), поэтому звук не продолжает играть после переключения.
- При запуске видео на слайде скрипт ставит на паузу другие медиа на странице (включая вне слайдера).

## Особенности

- **Обычные блоки Тильды**: в них нельзя назначить класс внутреннему элементу, только всему блоку. Поэтому для управления слайдером чаще используют отдельный zero-блок с кнопками.
- **Порядок блоков**: можно размещать элементы управления перед слайдами или внутри слайдов — оба варианта поддерживаются.
- **Последовательность слайдов**: блоки‑слайды лучше располагать подряд. Поведение при вставке других блоков между слайдами не тестировалось и оставлено для экспериментов и возможных доработок.
- **Видео в iframe**: для YouTube/Vimeo внутри iframe нет точного события `play` без включения API. Поэтому остановка других медиа срабатывает по клику/касанию по iframe.
